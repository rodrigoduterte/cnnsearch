# Purpose
To scrape news articles given a set of criteria. The user has the option to save articles and delete saved articles

# Routes
## / and /home
  These pages display the search, scraped articles and saved articles sections

## /saved
  This page displays the saved article section
  
# Sections
## Search
  The user must provide a valid topic, start year and end year before clicking the submit button. This section refreshes after submit button is clicked.
  
## Scraped Articles
  This section displays five articles after the submit button is clicked. Each article can be saved by clicking the saved button beside it.

## Saved Articles
  This section displays an unlimited number of saved articles. Each articles can be removed by clicking the remove button beside it.
